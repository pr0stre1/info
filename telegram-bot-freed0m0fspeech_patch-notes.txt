v 1.00

Изменения касающие пользователей:
    -Добавлены новые команды: /play, /skip, /clear, /pause, /now, /queue, /join, /leave, inline /play
    -Поддержка для команды /play: YouTube, SoundCloud, Twitch, Стримы, Видеовоспроизведение, Синхронизация видео-аудио
    -Поддержка для inline команды /play: YouTube, SoundCloud

Описание команд:
    /play [навзание] - добавить мультимедиа в очередь
    /play [навзание] @video@ - добавить видео в очередь
    /play [навзание] @video@ @sync@ - добавить видео в очередь с синхронизацией с аудио
    /play [навзание] @sc@ - добавить мультимедиа с SoundCloud
    /play [навзание] @yt@ - добавить мультимедиа с YouTube
    /play - начать воспроизведение мультимедиа из очереди
    inline /play [навзание] - поиск мультимедиа в Интернет
    /skip - Пропуск мультимедиа
    /clear - Полностью очистить очередь
    /clear [количество] - Очистить определенное количество из очереди
    /clear [-количество] - Очистить определенное количество из конца очереди
    /pause - Приостановить/Возобновить мультимедиа
    /now - Информация о текущем мультимедиа
    /queue - Очередь мультимедиа
    /queue [страница] - Определенная страница очереди
    /join - Присойденить бота в голосовой канал
    /leave - Отсоединить бота от голосового канала

Изменения в коде:
    -Поддержка GroupCall: handlers, events
    -Поддержка TwitchAPI: requests, handlers
    -Поддержка MongoDB: connection, requests
    -Поддержка Aiogram: bot, handlers, commands
    -Поддержка Pyrogram: bot, userbot, handlers, commands, inline commands
    -Поддержка WebServer: handlers, requests
    -Поддержка youtube_dl: search

Другие изменения:
    -Отображение информация во время воспроизведения
    -Отображения списка команд бота при вводе "/" в чате
    -Коректное отображение всей информации, ошибок и прочего
    -Сохранения очереди в Базу Даных
    -Изменения лицензии
    -Изменение структуры проекта

v 1.01

Небольшие изменения и фикс багов

Изменения касающие пользователей:
    -Добавлены новые команды: /speech_to_text, /lyrics, /stats
    -Поддержка для команд: /speech_to_text: GoogleRecognizer, /lyrics: Genius

Описание команд:
    /speech_to_text - необходтмо ответить на голосовое сообщение с командой /speech_to_text для распознания аудио
    /speech_to_text [код языка] (по умолчанию: ru-RU, Russian language) - распосзание других языков, кроме русского
    /lyrics - Слова песни для текущего мультимедиа
    /lyrics [навзание] - Слова песни для опеределенного названия песни
    /stats random - Рандомное сообщение групы
    /stats [учатсник] - Информация о участнике группы
    /stats - Топ участников групы

Изменения в коде:
    -Конвертор аудио (ffmpeg)

Другие изменения:
    -Поддержка Google Custom Search
    -Поддержка Google Search
    -Поддержка для Pyrogram v.1.3.1: изменение фильтров (группы, личные чаты)
    -Голосовой канал должен быть инициализорован для бота (end current voice group and create new one) если бот был
    оффлайн во время создания или завершения голосового канала
    -Количество результатов для inline команды /play изменено с 10 до 6

English

v 1.00

User changes:
    -Added new commands: /play, /skip, /clear, /pause, /now, /queue, /join, /leave, inline /play
    -Support for command /play: YouTube, SoundCloud, Twitch, Streams, video, sync
    -Support for inline command /play: YouTube, SoundCloud

Command description:
    /play [title] - add media to queue
    /play [title] @video@ - add media as video
    /play [title] @video@ @sync@ - add media as video with sync to audio
    /play [title] @sc@ - search and add media from SoundCloud
    /play [title] @yt@ - search and add media from YouTube
    /play - start playing from media queue
    inline /play [title] - search media via Ethernet
    /skip - skip current playing media
    /clear - full clear media queue
    /clear [count] - clear count of media from queue
    /clear [-count] - clear count of media from back of queue
    /pause - pause/resume playing media
    /now - info about current playing media
    /queue - media queue
    /queue [page] - page of media queue
    /join - join groupcall
    /leave - leave groupcall

Code changes:
    -GroupCall support: handlers, events
    -TwitchAPI support: requests, handlers
    -MongoDB support: connection, requests
    -Aiogram support: bot, handlers, commands
    -Pyrogram support: bot, userbot, handlers, commands, inline commands
    -WebServer support: handlers, requests
    -youtube_dl support: search

Other changes:
    -Show information when start playing media
    -Show list of bot commands by using "/" in chat
    -Properly output all info, errors, success, fails
    -Saving queue of media to DataBase
    -License changes
    -Project structure changes

Minor improvements and bug fixes

User changes:
    -Added new commands: /speech_to_text, /lyrics, /stats

    created new or ended previous voice group
    -inline /play results count changed from 10 to 6
